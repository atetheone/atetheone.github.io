Chat app

A full stack chat application that allows you to interact with others and provides a simple interface to chat with connected users. The user has to be authenticated to access to the chat feature. The chat feature is available in real time thanks to the Socket.IO. Microshop - Microwarehouse app

A set of two applications: a microwarehouse and a microshop. The first app is a microwarehouse application, which efficiently manages the addition and retrieval of products from the warehouse based on customer orders. The second app, a microshop application, provides a seamless ordering experience for customers, allowing them to easily select their desired products and track their order status.
Both of these applications boast a modern and intuitive user interface, thanks to the implementation of Angular on the front-end. On the back-end, I utilized NestJS to ensure robustness and scalability. One notable feature of these apps is the integration of an event-driven architecture on the backend side. This means that every action performed by users triggers an event, which is automatically published for listeners to subscribe to. This architecture enhances efficiency and allows for seamless communication between different components. To ensure the reliability and quality of these applications, I employed Cypress for end-to-end testing. This powerful tool ensures that all functionalities are thoroughly tested and guarantees a smooth user experience.

Random users generator

This app is built using Angular and Bootstrap. It serves as a platform to retrieve random users from the Random user generator API. The app allows users to delete existing users or add new ones . The user interface of the app is designed using Bootstrap, ensuring a responsive and visually appealing layout. Users can view details such as name, email, phone number, and address for each randomly generated user. The app also provides functionality for deleting users. Users can select any user from the list and remove them from the display. This action update the local array where users generated are stored in. Users have the ability to generate new users and add them to list to the list. Additionnally, the user can performe search action on the list, sort users by age, or filter by gender. To optimize performance and improve search engine optimization (SEO), Angular Universal is implemented in this app. Angular Universal enables server-side rendering (SSR), which pre-renders pages on the server before sending them to clients.
